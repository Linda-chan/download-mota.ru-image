AJPapps - Download mota.ru image

Линда Кайе 2017-2018. Посвящается Ариэль
=====================================================================

Этот скрипт скачивает обойки с сайта mota.ru. Достаточно передать в качестве параметра URL страницы с обойкой (приходят в рассылке), скрипт же скачает оригинал картинки и сохранит в текущий каталог.

Скрипт этот переделан из AJPapps - Download ftop.ru image, который переделан из AJPapps - Bing logger, который, в свою очередь, переделан из AJPapps - Nuerburgring logger ^^

Этот скрипт по историческим причинам имеет две версии:

1. WHT ver. Использует WinHTTP Services. Работает только в системе Windows 2000 и выше. При этом ни чего дополнительно ставить не нужно. Не умеет сохранять URL скаченной картинки в файл Descript.ion.

2. HTR ver. Использует мой собственный WinInet класс - ajpHTTPRequest. Поэтому требует библиотеки CommonFunctions0301.DLL. Её можно установить с любой моей программой. Однако работает даже в Windows 98.

Кроме того каждая версия имеет две подверсии:

1. Консольная. Предназначена для запуска с CScript.EXE. Сообщения об ошибках выводит в консоль. Так же выводит сообщения о каждом шаге скачки.

2. Графическая. Предназначена для запуска с WScript.EXE. Сообщения об ошибках выводит через стандартные окна сообщений. Других подробностей она не сообщает.

Вот пример использования скрипта:

> Download-mota.ru-ImageHTR.WSF http://www.mota.ru/wallpapers/view/id/51850

Обратите внимание на сохранение картинки в текущий каталог.
